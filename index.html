Heroes of Might and Magic III

Стек Python (бэкенд), Oracle (база данных/логика игры), HTML/CSS/JS (фронт).

5. Версия 0.5

URL: Rutube

В версии 0.5 был реализован наём войск: мечников, лучников, кавалеристов. У шахт и лесопилок теперь есть функция уровней. Также был реализован замок Чёртов, который можно захватить с помощью своих войск.

4. Версия 0.4

URL: Rutube

Версия 0.4 включает в себя наличие шахт и лесопилок, которые приносят ресурсы. С каждым днем ресурсы увеличиваются. Реализована мини-карта, на которой отображаются исследованные/не исследованные территории.

3. Версия 0.3

URL: Rutube

Версия 0.3 включает в себя наличие счетчика ходов, реализованы дни. Каждый день предоставляет игроку использовать 20 шагов. При ходьбе по траве, лесу, мосту отнимается 0.5 шага, при ходьбе по пустыне: 1 шаг.

2. Версия 0.2

URL: Rutube

Версия 0.2 включает в себя улучшения в интерфейсе пользователя, карты, а также исправления обнаруженных ошибок в версии 0.1. GUI представлен в виде веб-сайта.

1. Версия 0.1

URL: Rutube

Версия 0.1 представляет собой начальный этап разработки. В этой версии реализованы базовые механики игры и основная структура проекта. GUI представлен в командной строке.
